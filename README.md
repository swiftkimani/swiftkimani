### 🏢 [Asset Nexus](https://github.com/swiftkimani/asset-nexus)
**Enterprise-grade asset lifecycle management system.**  
Built with `Next.js 15`, `TypeScript`, `SQLite (Turso)`, and `shadcn/ui`. Track, assign, report, and manage hardware/software assets with role-based access control, bulk import/export, and a modern responsive dashboard.  
> 🌟 *Why it's awesome:* It's a full-stack, enterprise-ready SaaS template showcasing complex data modeling, strict access control, and a sleek modern UI.

### 🤖 [Social Scheduler](https://github.com/swiftkimani/social-scheduler)
**AI-powered social media scheduling agent.**  
A high-performance backend built with `Go` paired with a lightning-fast `Next.js` frontend.  
> 🌟 *Why it's awesome:* Seamlessly integrates AI workflows with Go's concurrency to handle social media automation reliably.

### 🛒 [Nextjs-Ecommerce](https://github.com/swiftkimani/Nextjs-Ecommerce)
**Modern Headless E-Commerce Platform.**  
A fast, SEO-friendly e-commerce application utilizing `Next.js` for server-side rendering and static generation.  
> 🌟 *Why it's awesome:* Delivers a native-like shopping experience on the web with optimized image loading and state management.

### ⚡ [Golang Backend APIs](https://github.com/swiftkimani/goolang-backend)
**Scalable Backend Microservices.**  
A robust backend architecture built using `Go` emphasizing clean architecture, high throughput, and memory efficiency.  
> 🌟 *Why it's awesome:* Demonstrates strong backend engineering principles, RESTful API design, and performance optimizations.
